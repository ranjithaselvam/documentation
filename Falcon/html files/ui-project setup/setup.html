Setup Falcon UI Project

Following are the steps to start Falcon-UI project for first time:

  • Step 1 : Open Eclipse.
  • Step 2 : Click on File menu and select New from given list then click on Maven Project.

  • Step3 : Click Next under Maven Project.

  • Step 4 : Click Configure.

  • Step 5:Click on Add Remote Catalog

  • Step 6 : Enter below details under Remote Archetype Catalog window and click Ok.
    • Catalog File : FALCON
    • Description : It can be any name to identify the Catalog say Falcon-Catalog.

  • Step 7 : After Adding Remote Catalog It will show as Remote: NameWhichYouGave on Archetypes Window
  • Step 8 : Then click Apply and Close to save changes(after saving changes it will show all archetypes of falcon ).

  • Step 9 : UnCheck the option called “Show the last version of Archetype only ”(it will show you all old and new archetypes of falcon).

  • Step 10 : Select Falcon-UI Archetype (with any version whatever we want eg . 0.1.0, 0.2.0 etc)as Archetype ID and then ClickNext.
