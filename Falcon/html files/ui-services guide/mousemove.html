3. public boolean dragAndDropBy(WebElement element, int x ,int y)

  • Method Name:

    • public boolean dragAndDropBy(WebElement element, int x ,int y)

  • Purpose:
    • This method performs click and hold the source element then move it to destination element by X and Y offset.
    • You will have to pass in the element that you want to move and then the x and y offset that you want to move it by.
  • Parameters:

    • This method having three Parameters:

    • WebElement- It holds identification of web element.
    • Offset X and Offset Y- X and Y are padding value of web element where we have to move the element.
  • Returns:

    • It returns true if drag and drop successful otherwise returns false.

  • Exception :

    • NullPointerException
