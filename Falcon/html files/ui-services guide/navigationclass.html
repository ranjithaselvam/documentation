1.public boolean back()

  • Method Name:

    • public boolean back()

  • Purpose:
    • This method is use to navigate the page to back.
    • It will navigate page in same window which is open, It will not open another browser or another window to show the back page.
  • Returns:

    • It returns true if back page will shown successfully otherwise it returns false.

  • Uses:


2. public boolean forword()

  • Method Name:

    • public boolean forword()

  • Purpose:
    • This method is used to navigate to the next page.
    • It will navigate to the next page in same window which is open, It will not open another browser or another window to show the next page.
  • Return:

    • It returns true if page successfully navigate to next page otherwise returns false.

  • Uses:


3. public boolean refresh()

  • Method Name:

    • public boolean refresh()
