In addition, you can also email us directly at support@falcon.com if you would prefer.

What is FAT?

FAT stands for Falcon - Automation Tool.

What is Falcon?

The Falcon UI Automation Framework is a tool for testing web applications. This framework can be used for automating most routine processes in web application, such as:

  • Cross browser Testing, Execute on Local machine, remote machine(grid), Cloud (BrowserStack).
  • Verification of different data types(Primitive and Non-Primitive).
  • Utilities to read and write with different file formats(.xls, .xlsx, .txt, .csv, .xml).

What are the tools and frameworks used by the FAT?

FAT uses the following:

  • Java - Programming language
  • TestNG - Unit Framework
  • Maven - Build Management Tool
  • Selenium - Automation Tool
